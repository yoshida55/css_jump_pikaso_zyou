# コーポレートサイト（ハウスメーカー）コーディング仕様

## 基本設定

| 項目 | 値 |
|------|-----|
| コンテンツ幅 | 固定サイドメニュー：300px<br>メインコンテンツ最大幅：900px<br>横のパディング：20px |
| ブレイクポイント | 960px |
| フォント | Google Fonts「Noto Sans JP」 |

---

## 各セクション仕様

### Header（サイドメニュー）
- 幅：300px固定（レスポンシブ時も伸縮しない）
- `position: fixed` で固定
- ロゴ（h1タグ）とグローバルナビ（nav > ul > li）を配置
- グローバルナビとSNSリンクを含む
- SP時：ハンバーガーメニューに切り替え、ヘッダーを上部に固定

### 動画エリア（#video）
- 幅：全幅（100%）
- `video` タグを使用
- 背景に全幅で固定

### 建築事例をご紹介（#project）
- コンテンツ幅で中央配置（共通クラス `wrapper` を使用）
- タイトル：h2タグ、疑似要素で両サイドに線
- 画像：ul、liで横並び（Flexbox使用）
- 画像の白枠線：CSSで設定
- 画像内テキスト：`position` で画像上に重ね、`box-shadow` で黒い影
- 背景：徐々に暗くなるグラデーション透過を設定
- 画像全体をリンクに設定

### 明るいHOUSEの特徴（#feature）
- コンテンツ幅で中央配置（共通クラス `wrapper` を使用）
- タイトル：h2タグ、疑似要素で両サイドに線
- 背景：透過
- 3つの円：ul、liで横並び（Flexbox使用）
- 円全体をリンクに設定
- ホバー時：左から右にスライドするよう背景色を変更
  - 疑似要素、`position`、`transition` を使用して実装

### 家ができるまでの流れ（#flow）
- コンテンツ幅で中央配置（共通クラス `wrapper` を使用）
- タイトル：h2タグ、疑似要素で両サイドに線
- フロー図：CSSで作成（※実務では画像を使用することが多いが練習のためCSSで作成）
- 流れ図全体：divタグで囲み、Flexboxで横並び
- 図形：順序性があるため `ol` タグを使用（`ul` ではない）
- 三角部分：liタグの疑似要素で作成、`position` で配置
- 黄色い棒線：olタグの疑似要素で作成、`position` で中央配置
- 説明：dl、dt、ddタグを使用
- 丸数字：dtタグ内でspanタグを使用（疑似要素でも可）

### お問い合わせ（#contact）
- コンテンツ幅で中央配置（共通クラス `wrapper` を使用）
- タイトル：h2タグ、疑似要素で両サイドに線
- 3つのボックス：divタグで囲み、Flexboxで横並び
- 各ボックスにリンクを設定
- 右下の黒三角：疑似要素で作成、`position` で配置
- ホバー時：枠線を太く（3px）する
  - **重要**: `border` ではなく `outline` を使用
  - `border` はレイアウトがずれるが、`outline` はボーダー領域を取らないためずれない

### Footer
- フッター全体を `footer` タグで囲む
- メニュー：ul、liで記述、Flexboxで横並び配置

### トップへ戻るボタン
- CSSで作成
- 右下に固定配置（`position: fixed`）
- 初期表示：jQueryの `hide()` で非表示
- スクロール位置が700pxを超えた時点で表示
  - jQueryのscrollイベントと `fadeIn()` を使用
- クリック時：`animate` でページトップへ戻る

## レスポンシブ対応

```css
/* PC */
@media (min-width: 961px) {
  /* PC用スタイル */
  /* サイドメニュー固定レイアウト */
}

/* SP */
@media (max-width: 960px) {
  /* SP用スタイル */
  /* ハンバーガーメニュー */
  /* ヘッダー上部固定 */
}
```
